# Link

## Introduction

Link actions can be used to redirect users to other pages or absolute URLs. They are particularly
useful for navigation or redirecting. Supported link actions are the following.

- [`link`](../reference/action.mdx#link)
- [`link.back`](../reference/action.mdx#link.back)
- [`link.next`](../reference/action.mdx#link.next)

## Actions

### link<Side name="link" />

<ActionRef action={referenceSchemas.linkActions['link']} />

Here is how you can use this action in your app.

```yaml validate blocks-snippet
blocks:
  - type: action-button
    version: 0.34.1-test.2
    parameters:
      icon: circle-chevron-right
    actions:
      onClick:
        type: link
        to: Page 2
```

### link.back<Side name="link.back" />

<ActionRef action={referenceSchemas.linkActions['link.back']} />

The following is an example of this action being used in an app.

```yaml validate blocks-snippet
blocks:
  - type: control-buttons
    version: 0.34.1-test.2
    parameters:
      back: { icon: chevron-left }
      forward: false
    actions:
      onBack:
        type: link.back
```

### link.next<Side name="link.next" />

<ActionRef action={referenceSchemas.linkActions['link.next']} />

The following is an example of this action being used in an app.

```yaml validate blocks-snippet
blocks:
  - type: control-buttons
    version: 0.34.1-test.2
    parameters:
      back: { icon: chevron-left }
      forward: { icon: chevron-right }
    actions:
      onBack:
        type: link.back
      onForward:
        type: link.next
```
